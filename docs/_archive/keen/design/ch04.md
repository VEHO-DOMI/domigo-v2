## 4 · Level design (build spec — cookbook §7 archetype "switch-box dungeon")
- Size ~60×33 (4×3 screens), boxed interior floors with machine-state routing.
- Flow: workshop entry (rolling gear-floor, slopes as ramped cog-teeth) → mood-machine
  teach (one switch, one platform set flips happy⇄sad state — feelings = states, the
  motif) → door `1` → **Lagerraum wing** (parts storage, seal 1, walker guard) → chain
  poles up the gear shaft (slide-down = fall recovery) → the flipped hall (two machine
  states alternate the one-way route — read the mood-light, then move) → door `2` →
  **Maskenkammer wing** (the mask store, seal 2, hopper guard) → piston mover (a lift
  crossing the boiler shaft — its one job) → roof vents → the guardian's door.
- Verbs: slopes on the cog-floor · poles as chains (ledges ≤1 row above pole tops) ·
  door pairs per wing · 1 mover (piston lift) · drop-through vent grilles.
- Population (E, ≤1/screen): walker (floor) ×1 · flyer (gear shaft) ×1 · hopper ×1 ·
  guards: walker in the Lagerraum (seal 1), hopper in the Maskenkammer (seal 2). M adds
  the signature sending — a Flatterklecks mask-variant whose color flips with the machine
  state (data twist); S adds a thief in the flipped hall.
- Collectibles: letters along the flipped hall's two states + wing shelves · 3 Glühwörter
  (boiler alcove, chain-shaft top, mask-store loft) · helpers ×2 E.
- Secret: drop-through behind the boiler → steam alcove (Glühwort) → pole back out.
