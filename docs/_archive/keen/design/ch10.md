## 4 · Level design (build spec — cookbook §7 archetype "shelf tower (up)")
- Size ~35×50 (2.3×4.5 screens), vertical UPWARD climb — the inverse of ch09's descent
  (the never-same-shape-twice rule holds).
- Flow: market square (lamppost + sorry-note dressing, slopes) → shop floor → **shelf terraces**
  (facade-as-level grammar: shelving as terraced steps, one-way shelf boards `=`) → **stock-lift
  mover 1** (crosses the open aisle shaft — the fall-recovery job) → **storeroom wing via door
  pair 1** (seal 1, walker guard) → higher shelf climbs → **stock-lift mover 2** (ferry to the
  display-window ledge) → **display-window wing via door pair 2** (seal 2, hopper guard) → the
  top counter (boss door at the till).
- Verbs: the vertical shelf climb uses **movers as stock lifts** (2, each with a job, period
  ≥2.6s) · one-way shelf boards `=` as the climb texture · door pairs per wing · 1 pole as the
  aisle-shaft recovery spine.
- Signature sending: **the Wortdieb, prominent** (existing wave-1 sprite, shop-thief recolor) —
  placed ×3 as the level motif on the Treasure-Eater grammar: it steals letters and flees to a
  nameable stash pocket, so every chase pays off (cookbook §7).
- Population (E, ≤1/screen): thief ×3 (signature) · walker ×1 · hopper ×1 · guards: walker in
  the storeroom (seal 1), hopper at the display window (seal 2). M adds a flyer on the terraces;
  S adds a cloud over the aisle shaft.
- Collectibles: letters breadcrumb the terraces + the thieves' stash pockets · 3 Glühwörter
  (stash pocket top, storeroom ceiling pocket, display-window ledge) · helpers ×2 E.
- Secret: a just-past-where-the-camera-seems-to-stop passage behind the counter → cellar
  alcove (Glühwort) → pole out.
