## 4 · Level design (build spec — cookbook §7 archetype "burrow warren (down)")
- Size ~40×48 (2.7×4.4 screens), vertical DOWNWARD warren — the campaign's first true descent.
- Flow: Tim's yard (perches, slopes, filled-bowl dressing) → **soft-drop shaft 1** (drop-through
  root platforms `=` stacked as the descent motif) → **rabbit warren via door pair 1** (seal 1,
  walker guard) → central gallery — the feed-bucket mover (a lift crossing the deep shaft, the
  fall-recovery job) — → **soft-drop shaft 2** → **bat hollow via door pair 2** (black-dominant
  interior, seal 2, flyer guard) → optional **budgie loft** (a pole climb back up half the map:
  letter-rich + 1 Glühwort — the optional wing) → the bowl-chamber (boss door).
- Verbs (the DOWN lean): drop-through `=` everywhere the descent needs softness · **poles as the
  climb-back spine** (slide down fast, hop off at wing mouths) · door pairs per burrow wing ·
  1 mover (feed-bucket lift, period ≥2.6s).
- **Wave note — spawner nests**: 2 nests in the bottom gallery on the Pea-Pod grammar (each
  spawns hoppers to a hard budget, **max 4 alive from nests total**, cookbook §7). Nests are
  actor-table data (spawn budget as a parameter), not a new engine verb — flag in the W-brief.
- Population (E, ≤1/screen): walker ×2 · hopper ×1 · flyer ×1 · guards: walker in the rabbit
  warren (seal 1), flyer in the bat hollow (seal 2). M adds a cloud over shaft 2 + nest budget
  +1; S adds a thief near the budgie loft.
- Collectibles: letters breadcrumb the shafts + cluster in the loft · 3 Glühwörter (loft top,
  behind the mover's far anchor, bat-hollow ceiling pocket) · helpers ×2 E.
- Secret: a shimmering wall gap behind a Hüpfer's arc (the fair hint) → hamster hollow
  (Glühwort) → drop-through exit.
