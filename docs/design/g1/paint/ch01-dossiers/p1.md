## 4 · Der Pfad

Prolog-Bilder → aufwachen (A) → Bank-Sprünge unter „S-C-H" (B) → Läufer-Begegnung (B) →
Halte-Sprung „O-O-L" (C) → Krakel + Brett (C) → Treppen-Grube: Buchdeckel-Treppe oder
Fall-Lektion (D) → Radier-Hüpfer-Timing (D) → Spind-Gasse: Käfig #2 rüttelt, Alkoven-Tease
im Licht (E) → „Come in!" → p2. **Verben: gehen, Tap-Sprung, Halte-Sprung, berühren/lösen.**

## 5 · Bild-Behandlung (VL 1.2b — Innenraum-Fassung)

- **Gelände = Schulhaus-Material:** Boden = DIELEN-Kruste über BÜCHER-Körper-Masse;
  Podeste = Garderoben-Bänke, Bücherstapel, das Treppen-Podest; Stufen = Buchdeckel;
  dünne Podeste = Papierbögen. KEIN Gras, KEINE Erde, nirgends (G1).
- **Ebenen:** FERN = die Hallen-Rückwand (hohe Fenster mit Vormittags-Licht, Garderoben-
  Haken-Reihe, Türen-Flucht — Honig-Shift); MITTE = Spind- und Bank-Silhouetten; NAH =
  Spielband; FG = Balken-Okkluder + Lampen-Kette. Oben: Decken-Balken schließen.
- **Affordanz:** Spinde = hohe Massen, Bänke = Podeste (flach=stehen), Tinte = Kill-Farbe,
  Kreide-Pfeile = Wegweiser. Verb-Schutt: Späne, Kreide-Krümel.
- **Augen-Gesetz:** Läufer, Hüpfer, Käfige, die Tür — ja; Spinde, Bänke, Brett — nie.
