## 4 · Level design (as built, v2.3)
- Archetype: **schoolhouse facade climb** (Moons pattern), 72×24 (4.8×2.2 screens).
- Flow: rolling yard (slopes `/` `\`) → flagpole teach → porch door `1` → **Klassenzimmer wing**
  (seal 1, walker guard) → facade terraces → roof deck → door `2` → **Dachboden wing** (ferry
  mover to the seal island, seal 2, hopper guard) → gutter spikes → roof gap (lift mover OR
  recovery pole — two routes) → tower → drop-through hatch secret (Glühwort alcove, pole
  return) → gable → the guardian's door.
- Population (E): walker yard · hopper porch · flyer facade · walker classroom-guard ·
  hopper attic-guard; M adds gap-walker + roof-flyer; S adds attic thief.
- Collectibles: 16 letters on movement arcs · 3 Glühwörter (yard shelf, classroom shelf,
  tower alcove) · helpers ×2 (tier E).
- Secret: the tower hatch (drop-through `=`) → alcove → pole top-out.
