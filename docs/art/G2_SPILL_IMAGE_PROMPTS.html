G2 · The Spill — sheet prompt library (ART-2 format)

The three pipeline rules (supersede the single-element workflow):
1 · One generation = one SHEET — a 4-column × 3-row grid of elements on solid magenta #FF00FF. Never ask the generator for transparency: transparency is enforced by our slicer, never trusted from the generator.
2 · Save the sheet PNG into the zone's drop folder, then run:
node docs/art/slice-art.mjs --sheet <sheet.png> --manifest docs/art/sheets/g2-<zone>.manifest.json → then the usual prep-art.mjs (QA/resize) + sync-art.mjs.
3 · Cell ORDER is the contract (row-major, matching the manifest) — the prompt never asks for text labels inside the image (text breeds artifacts).
Generator calibration (Koki's tools):
GPT-Image 2 — obeys grids well. Keep the GRID block verbatim at the top; if a cell bleeds, regenerate adding "each element fully inside its cell, generous magenta margin around every element."
Nano Banana Pro 2 — needs redundancy: repeat the background demand ("solid flat magenta #FF00FF everywhere between elements — NO checkerboard, NO gradient, NO texture") and prefer "12 separate small pixel-art objects arranged 4 across, 3 down" over abstract grid language. If it draws thin cell borders, fine — the slicer crops inside them.
