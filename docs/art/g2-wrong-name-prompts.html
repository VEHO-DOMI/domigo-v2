How to use · style block · character bible

How to use: click Copy and paste the whole prompt into ChatGPT / GPT-image as-is — the style and the relevant character descriptions are already baked into every prompt. For the strongest consistency, generate a character's full-body reference (Section A) first, then in ChatGPT attach that image alongside a scene prompt; keep to one main character per image where you can. All prompts are deliberately text-free (facts are shown visually — e.g. the result gap is a tall teal bar vs a short maroon bar).
