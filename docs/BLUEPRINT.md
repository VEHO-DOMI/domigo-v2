## Part II — North star & binding principles

**North star (the erase-the-teacher test, ages 10–14):** a Klasse 1–4 student can open DomiGo on a cheap phone and get from any starting point to A2+ mastery without Koki in the room — every concept taught before it is tested, every wrong answer turned into a transferable skill named in kid-German, every unit carried by story-pull, nothing above their level unglossed, and nothing they do able to lose their progress. Koki's class time converts to speaking and projects; parents can read honest progress in German.

**Binding principles (superset of the 10 content principles + 9 game Laws; executor sessions must not violate):**
1. MORE! 1–4 is the only content source; **verify-don't-trust** (generate → independent verify → deterministic validate → Koki-sample).
2. **Level gate over every student-facing string** incl. games/flavor/choices/teasers; du-form; zero meta/teacher-talk; glosses `(= deutsches Wort)`.
3. **ONE grading brain** (`@domigo/engine`); every economy is a re-skin of `xpForTier`; no second grader, no third XP pool; grading is forgiving AND correct (complete variant coverage).
4. **Progress derives from `practice_attempts` server-side**; `game_saves` cosmetic ≤64KB LWW; failure changes pace never position; timers only on mastered material (the fluency-arcade exception).
5. **Spaced retrieval is world physics** — `getDueRefs` powers review, encounters, bosses, and the re-fog surfaces; review disguised as play.
6. **Additive-only `domigo_v2` DDL**; never touch `public.*`; one migration per PR, applied to the dev branch before merge; prod DDL only via the D-2 runbook sitting.
7. **Bulletproof:** checkpoints ≤90s, offline outbox, no dead ends, class-scale-tested, no dead toggles.
8. **Content is regenerable** — fix at canonical source or the git overlay (`content/overlays/item-fixes.json` / `sibling-fixes.json`), never hand-edit generated JSON; the Studio's DB overlay folds back to the overlay files, never to generated corpus.
9. **Student-data-first (minors):** DSGVO posture, pseudonymous students, teacher-only real names, AI-pipeline + telemetry disclosure before any released AI output (D-6).
10. **The blueprint lives in-repo**; code outranks docs; STATUS reconciled ≤3 merges behind; plans-dir and session memory are volatile scratch, never authoritative.
11. **Koki's gates are sacred:** he squash-merges every PR; register/voice gates precede every authoring wave; blind-solve at 0 disagreements on key changes; fixed sampling tiers; never a bare `#N` in a commit message (auto-closes that PR).

### III.4 · WS-G4 — the fourth game: **"Lost for Words" / "Sprachlos"** (`g4.st.lost-for-words`, package `@domigo/game-trip`)

_The narrative was redesigned this session after Koki rejected all three fantasy options. The mechanics layer below ("the locked layer") is narrative-agnostic, repo-verified, and proceeds in parallel with the G4-N narrative gate._

**Logline.** On your class's England intensive week (Sprachwoche — a real 4.-Klasse AHS tradition), your best friend Niko is quietly blamed for a ring that vanished from your host family's kitchen — and the worst sentence of the week is the one you didn't say.

**Why it's fresh vs G1/G2/G3 (one line):** G1 restored a magic book, G2 solved a crime, G3 watched the internet hurt a friend — G4 is the first game where the player's own English, produced under real social pressure in a real place, is both the setting and the moral instrument: abroad, every task is survival, and the harm is done not by a villain but by retellings and a silence — yours.

**Central conflict + fair-play design (no culprit exists).** Night two, Nan's ring — the "something old" for Katie's wedding — disappears from the kitchen. Niko had admired it at dinner. By the youth-club evening, "he was **holding** it" has become "the Austrian **took** it" — polite, deniable sentences in a language the boys can barely fight back in. The truth is a pure accident, reconstructable from Fragments planted W2–W7: ring came off during the cooking (F1) → onto the windowsill (F3) → counters cleared into the charity bags (F4) → one box scooped into host-dad Dev's **auction** lot by mistake (F5, the mislabeled receipt) → the ring **turns up** in a bric-a-brac lot at the Saturday auction (THE TURN, W8). The rumor chain is also fair-play: Lewis's original sentence is on record from W3, and the player watches each retelling drift. Every link is innocent-ish and human; the harm emerges from the system — G2 inverted: there a theft turned out to be a correction; here there is no theft at all, only sentences.

**Corpus fit (verified against the actual g4 wordbanks — 12–13 of 13 units native):** u01 fluent/foreigner/incident/shake hands (+ the Ireland set via Irish Nan's stories) · u02 crime scene/evidence (spent through Callum's true-crime parody, never the narration) · u03 announcement/crowd · u05 cookery/tasty/revolting/trust/(be) ashamed/even though · u06 community/donate/grateful · u07 aircraft/airline/timetables · u08 **auction/collection/precious/rare/turn up/whisper/furious/pale/judge** · u09 the complete wedding lexicon + giggle/embarrassed/goth/pierced/rebellious (cousin Ash) · u10 **fairness/helpless/agreement/claim** · u11 anthology/biography/fiction/blurb · u12 astronaut/crew/commander (the real **Leicester National Space Centre** — a genuine Sprachwoche excursion) · u13 **apply (for)/attend/best wishes/highlight/honestly/last but not least**.

**(a) SESSION BOOT** — every session, no exceptions: `cd ~/Code/domigo-v2 && git checkout main && git pull`; `git status` clean; `gh pr list --state open` (expect 0 or the one in-flight).
